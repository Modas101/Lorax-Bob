# Design Instructions

## Background Image

To complete the UI redesign, please add the Bright Path background image:

1. **Save the Bright Path logo image** (the one with mountains and blue gradient) as:
   ```
   public/bright-path-bg.jpg
   ```

2. **Image specifications:**
   - Format: JPG, PNG, or WebP
   - Recommended size: 1920x1080 or larger
   - The image should show the mountains at the bottom portion

3. **Current placeholder:** 
   - A placeholder file exists at `public/bright-path-bg.jpg`
   - Replace it with the actual Bright Path background image

## UI Layout

The new design features:
- **Background:** Full-screen Bright Path image with mountains
- **Chat Interface:** Floating white card in the upper-middle area with black/white color scheme
- **Input Box:** Separate white card at the bottom between the mountains
- **Color Scheme:** Black and white for chat UI to stand out from the blue backdrop
  - User messages: Black background with white text
  - AI messages: Light gray background with black text
  - All UI elements use clean black/white/gray tones
